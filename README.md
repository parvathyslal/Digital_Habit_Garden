# Digital Habit Garden

## Project Overview

- Digital Habit Garden is an interactive habit tracking web application.
- The project is designed to make daily habit building more engaging through a virtual garden concept.
- Users can create a profile, choose a theme, track habits, use productivity tools, and view their progress through a visually rich interface.
- The application combines habit tracking, focus management, streak monitoring, and animated UI elements.
- The project focuses on frontend development, responsive design, user interaction, and creative UI/UX implementation.

## Objectives

- To build a habit tracking platform with an engaging visual experience.
- To encourage users to maintain daily consistency.
- To represent personal growth through a garden-based interface.
- To provide a clean and responsive multi-page website.
- To improve frontend development skills using HTML, CSS, and JavaScript.
- To create an interactive project suitable for portfolio and academic presentation.

## Pages Included

- `home.html`
  - Main landing page of the project.
  - Introduces the Digital Habit Garden concept.
  - Provides navigation to other pages.

- `login.html`
  - Allows users to create a garden profile.
  - Includes name, avatar, theme, and goal selection.
  - Stores user preferences using local storage.

- `about.html`
  - Explains the purpose and idea behind the project.
  - Describes how habit tracking is connected with virtual garden growth.
  - Presents the design philosophy of the application.

- `features.html`
  - Lists the main features of the application.
  - Highlights habit tracking, themes, focus timer, streaks, and progress visualization.

- `garden.html`
  - Displays the interactive virtual garden.
  - Shows garden growth based on user progress.
  - Includes animated and theme-based visual elements.

- `dashboard.html`
  - Provides an overview of user progress.
  - Displays habit-related data and productivity information.
  - Acts as the central progress monitoring page.

- `contact.html`
  - Provides contact or developer information.
  - Serves as the communication section of the project.

## Features

- Multi-page frontend web application.
- Interactive habit tracking system.
- Virtual garden growth concept.
- User profile creation.
- Theme selection support.
- Avatar and goal selection.
- Dashboard for progress overview.
- Focus and productivity-based interface elements.
- Streak-based motivation system.
- Animated visual effects.
- Responsive layout for different screen sizes.
- Local storage support for saving user data.
- Modern glassmorphism-inspired UI design.
- Smooth transitions and interactive hover effects.

## Technologies Used

- HTML5
  - Used for structuring the web pages and application layout.

- CSS3
  - Used for styling, animations, glassmorphism effects, responsive layouts, and theme design.

- JavaScript (Vanilla JavaScript)
  - Used for DOM manipulation, interactivity, theme switching, habit tracking logic, local storage handling, and animations.

- React.js
  - Used in the Garden module for rendering dynamic and interactive UI components.

- HTML5 Canvas API
  - Used for animated ambient backgrounds and environmental visual effects.

- Browser Local Storage
  - Used to store user preferences, habit progress, streak data, themes, and profile information locally.

- Responsive Web Design
  - Implemented using media queries and flexible layouts for compatibility across desktop and mobile devices.
